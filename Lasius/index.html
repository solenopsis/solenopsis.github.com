asius

Java utility framework for SFDC

Lasius is a utility framework for simplifying connectivity to Salesforce custom web services as well as the metadata API, and enterpise/partner web services in Java.

Dependencies

  • Java 1.6
  • FlossWare Utils

License

GPLv2

Authors
